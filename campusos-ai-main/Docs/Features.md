# Features

*(Reflects what's actually implemented as of v1.0 — see CHANGELOG.md for how each feature evolved.)*

## Authentication
- Login, Signup, JWT sessions, protected routes, rate-limited auth endpoints

## Dashboard
- Real stats (task/note counts), quick actions, recent notifications, recent activity, upcoming tasks

## Notes
- Create, read, update, delete, search — plain text (no file upload/download)

## Tasks
- Create, read, update, delete, search, status filter, sorting, due dates

## Notifications
- Auto-generated from task/note activity, header dropdown + full page, mark read/all-read, unread badge (polling)

## Activity Timeline
- Dashboard preview + full timeline page — currently sample data, not yet backend-driven

## AI Assistant
- Chat UI (typing indicator, auto-scroll, message history) — preview only, no AI backend yet

## Profile
- View and edit name/branch/year, logout
