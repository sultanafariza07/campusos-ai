# 🎓 CampusOS AI

> The Operating System for College Life

CampusOS AI is a mobile-first student productivity platform designed to help college students manage notes, tasks, assignments, and AI-powered assistance in one place.

---

## 🚀 Features

### Authentication
- Login
- Signup
- JWT Authentication

### Dashboard
- Student Dashboard
- Quick Actions
- Recent Activity

### Notes Module
- Upload Notes
- Search Notes
- Download Notes

### Tasks Module
- Create Tasks
- Mark Complete
- Due Dates

### AI Assistant
- AI Chat Support

### Profile
- Edit Profile
- Logout
