# Software Requirement Specification

# Project Name

CampusOS AI

# Problem Statement

Students use multiple apps for notes, tasks, and academic activities. CampusOS AI aims to provide everything in one platform.

# Objectives

- Simplify student life.
- Centralize academic resources.
- Provide AI-powered assistance.

# Users

- College Students

# Functional Requirements

- Login
- Signup
- Dashboard
- Notes Management
- Tasks Management
- Profile Management
- AI Assistant

# Non-Functional Requirements

- Mobile First
- Responsive UI
- Secure Authentication
- Fast Performance
